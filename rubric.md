# Grading Rubric

## Assignment: Hands-on L2: Basics of GitHub

**Total Points: 1**

---

### Grading Criteria

#### GitHub Repository Submission (1 point)
- **1 pt**: Valid GitHub repository link submitted with appropriate content demonstrating understanding of GitHub basics
- **0 pts**: Missing submission, invalid link, or empty repository

---

## What to Look For
1. **Valid GitHub URL**: Link points to an accessible public repository
2. **Repository Structure**: Has at least a README or basic file structure
3. **Evidence of Git Usage**: Shows commits, proper repository initialization

---

## Grading Notes
- This is a pass/fail style assignment (0 or 1 point)
- Late submissions: Check "Days Late" field - instructor discretion on late penalty
- Submission type: URL to external GitHub repository
